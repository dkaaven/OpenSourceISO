# 0.3.1 De som har forventninger  
*(Interessenter – de vi må forholde oss til)*

## Hva handler dette om?

Alle virksomheter har mennesker eller grupper rundt seg som har forventninger.  
Dette kan være kunder, ansatte, myndigheter, samarbeidspartnere, eiere – og til og med samfunnet rundt oss.

Vi kaller dem **interessenter** – de som påvirker oss, og som vi påvirker tilbake.

---

## Hvorfor er dette viktig?

Når vi vet hvem som følger med på oss – eller er avhengige av oss – kan vi:
- Forstå behov og krav tidlig
- Ta bedre valg
- Prioritere riktig
- Bygge bedre relasjoner

---

## Eksempler på interessenter

| Gruppe | Eksempler |
|--------|-----------|
| Kunder | Forventninger til pris, kvalitet, levering, service |
| Ansatte | Ønsker trygghet, utvikling, respekt og retning |
| Myndigheter | Har krav til drift, rapportering og sikkerhet |
| Eiere eller styre | Ønsker lønnsomhet, kontroll og bærekraft |
| Leverandører og partnere | Forventer pålitelighet og samarbeid |
| Samfunnet | Har forventninger til miljø, ansvar og etikk |

> 🧠 Interessenter kan være både interne og eksterne.

## Tips

- Start med de viktigste – du trenger ikke fange “alle”
- Spør ansatte: Hvem merker vi forventninger fra?
- Dette gir grunnlaget for neste steg: **Hva de forventer av oss**
